Funcionamento

  • O projeto como se vê nas datas tem 2 revisões pré-entrega para que os alunos possam estar num bom caminho.
  • Após a primeira entrega os alunos que tenham confiança no seu trabalho e que achem que essa entrega seja a final não necessitam da segunda entrega.
  • Para os alunos que necessitem de mais tempo existe ainda outro deadline para a segunda entrega do projeto (isto afetará na entrega dos prémios)

Datas importantes

  • 15/04/2024 -> 1º Revisão do projeto
  • 15/05/2024 -> 1º Entrega do projeto
  • 01/06/2024 -> Saída dos resultados
  • 02/06/2024 -> Cerimónia de entrega dos prémios

Os avaliadores do projeto vão ser:

  • O professor que vos acompanhou durante o projeto
  • Professor Ricardo Silva
  • Professora Maria Ana
  • Professor João Mario Benfiquista

Prémios do concuros:

  • 1º Lugar -> Viagem para 2 para um destino dentro da EU por 7 dias
  • 2º Lugar -> Viagem para 1 para um destino dentro da EU por 3 dias
  • Todos os participantes receberam um diploma de participação e um brinde por terem acabado a licenciatura no iscte
